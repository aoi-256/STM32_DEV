サーボモーターを動かしてみよう

今回はPWMという新しい機能を使ってサーボモーター動かしてみる

今回から少しだけ配線が必要になるので、不安なら先輩に確認してもらおう

今回やること

作業の流れ

  • ピンを割り当てる
  • wrapper.cppにコードを書き込む
  • サーボモーターとSTM32を配線でつなぐ
  • STM32にプログラムを書き込み実行する

ピンの割り当て

今回はPWMという機能を使います

(プロジェクト名).iocからPA8を選択し、TIM1_CH1を割り当てよう

TIM1の設定からChannel1にPWM Generation CH1を割り当てよう

下のConfigurationからParameter Settingsを開き詳細な設定をしよう

Parameter Settings

各項目を以下のように設定してね

  • Prescaler
    83
  • Counter Mode
    UP
  • Counter Period
    2499
この設定でやっていること
  • タイマー設定

    サーボモーターは、パルス信号（周期的にオン・オフが切り替わる）の周期で角度を指定している

    この設定をすることで、STM32のクロック（計算処理とかをしている部分）をサーボモーターに送りたい周期に変換している

回路

サーボモーターとSTM32をジャンパー線を使ってつなぐ

配線を間違えるとサーボモーターが壊れてしまうことがあるので気を付けてください

真ん中の5Vだけは絶対に間違えないようにしよう！

プログラム

今回はタイマーのスタートと値の設定の2つの関数を使用します

TIMのスタート

指定したタイマーを起動するための関数

HAL_TIM_PWM_Start(&htim1, TIM_CHANNEL_1);

この関数の引数

  • &htim1
    使用するTIMの種類・・・TIM1を使用
  • TIM_CHANNEL_1
    使用するTIMのチャンネル・・・CH1を使用

TIMの設定

起動したタイマーの値を設定するための関数

__HAL_TIM_SET_COMPARE(&htim1 , TIM_CHANNEL_1, Value);

この関数の引数

  • &htim1
    使用するTIMの種類・・・TIM1を使用
  • TIM_CHANNEL_1
    使用するTIMのチャンネル・・・CH1を使用
  • Value
    設定したい値(0~65535),
TIMを止める関数

今回は使用しないが、安全装置などの設計をする上で止める関数が使えるので紹介しておく

HAL_TIM_PWM_Stop(&htim1, TIM_CHANNEL_1);

引数はスタートの関数と同じなので頭の片隅においておくといいかも?

サンプルコード

実際にサーボモータを回し続けるコードを作成した

0~2500の好きな値を指定できるので、色々な値を試してみよう

#include "wrapper.hpp"
#include "tim.h"

void init(){
    
    //タイマーの起動
    HAL_TIM_PWM_Start(&htim1, TIM_CHANNEL_1);
    HAL_Delay(500);     
}
        
void loop(){
    
    //値の設定
    __HAL_TIM_SET_COMPARE(&htim1, TIM_CHANNEL_1, 500);
    HAL_Delay(100-);
        
    __HAL_TIM_SET_COMPARE(&htim1, TIM_CHANNEL_1, 1000);
    HAL_Delay(1000);
        
    __HAL_TIM_SET_COMPARE(&htim1, TIM_CHANNEL_1, 1500);
    HAL_Delay(1000);
}
